    Class WfSignature

    A class representing a Whiteflag authentication signature WfSignature

    Wfversion

    v1-draft.7

    Wfreference

    5.1.2.1 Method 1: URL Validation

    Remarks

    Whiteflag uses JSON Web Signatures (JWS) as the structure for digital signatures used for authentication method 1. The Whiteflag digital authentication signature must be published at the URL where the A1 authentication message points to.
